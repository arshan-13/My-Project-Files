## Vrinda Store Sales Analysis

An end-to-end Excel data analysis project covering data cleaning, transformation, and interactive dashboard creation to analyze sales performance, customer demographics, and order trends for Vrinda Store, a multi-channel e-commerce retailer.

## Project Overview

The goal of this project was to analyze 30,000+ raw order records from Vrinda Store, clean and structure the data in Excel, and build an interactive Pivot Table/Pivot Chart-based dashboard to uncover sales trends, customer demographic patterns, and channel/regional performance — supporting data-driven business decisions.

## Tech Stack

| Tool | Purpose |
|------|---------|
| **Microsoft Excel** | Data cleaning, Pivot Tables, Pivot Charts, Slicers, dashboard design |

## Dataset

The raw dataset contains **31,000+ order records** with fields including Order ID, Customer ID, Gender, Age, Age Group, Order Date, Month, Order Status, Sales Channel, SKU, Category, Size, Quantity, Amount, and shipping location (city, state, postal code, country).

## Workflow

### 1. Data Cleaning & Preparation (Excel)
- Loaded the raw order-level dataset into Excel.
- Cleaned and standardized fields such as Gender, Age Group, Order Status, and Channel for consistent grouping and analysis.
- Structured shipping location fields (city, state, country) to enable regional sales analysis.
- Prepared the cleaned dataset as the source for Pivot Table-driven analysis.

### 2. Dashboard Design (Excel Pivot Tables, Charts & Slicers)
- Built multiple **Pivot Table** views to summarize the data across different dimensions:
  - **Age & Gender** breakdown of order volume
  - **Sales Channel** distribution (Amazon, Flipkart, Myntra, Ajio, Meesho, Nalli, Others)
  - **Men vs. Women** total sales comparison
  - **Order Status** breakdown (Delivered, Cancelled, Returned, Refunded)
  - **Top States** by sales amount
  - **Monthly Order vs. Sales** trend analysis
- Converted key pivot summaries into **Pivot Charts** for visual storytelling.
- Added **Slicers** to allow interactive filtering across the dashboard by channel, gender, age group, and order status.
- Consolidated all views into a single interactive **dashboard report** sheet.

## Key Insights
- **Channel performance:** Amazon was the dominant sales channel (~35% of orders), followed by Myntra (~23%) and Flipkart (~22%).
- **Gender split:** Women customers generated significantly higher total sales (₹13.5M+) compared to men (₹7.6M+).
- **Order fulfillment:** Of 31,000+ orders, the vast majority (~92%) were successfully delivered, with cancellations, returns, and refunds making up a small share.
- **Regional performance:** Maharashtra, Karnataka, and Uttar Pradesh emerged as the top revenue-generating states.
- **Demographics:** Adult women represented the largest customer segment by order share, indicating a key target audience for marketing focus.
- **Monthly trends:** Sales and order volume stayed relatively stable across the first half of the year, with March showing the strongest performance.
